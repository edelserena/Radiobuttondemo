# RadioButtonDemo - Java Assignment

A Java Swing application that demonstrates the use of `JRadioButton`, `ButtonGroup`, and `BorderLayout`.

## Features

- Select from 5 pets: Bird, Cat, Dog, Rabbit, and Pig.
- Images update dynamically based on your selection.
- All images are automatically scaled to fit the window (300x300 pixels).
- A message dialog pops up to confirm your choice.

## How to Run

1. Make sure you have the JDK installed.
2. Ensure the `.jpg` image files (`bird.jpg`, `cat.jpg`, etc.) are in the same folder as the `.java` file.
3. Open your terminal/command prompt in this folder.
4. Compile the code:

   ```bash
   javac RadioButtonDemo.java
   ```

5. Run the application:

   ```bash
   java RadioButtonDemo
   ```
